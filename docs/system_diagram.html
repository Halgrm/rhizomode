3. Edge Connection State Machine (Rector Style)

stateDiagram-v2
  [*] --> Idle
  Idle --> SourceSelected : Trigger on node\n(auto-select nearest output port)
  SourceSelected --> Idle : Trigger on empty / same node\n(cancel)
  SourceSelected --> SourceSelected : Trigger on incompatible node\n(stay, try another)
  SourceSelected --> Idle : Trigger on compatible node\n(auto-select input port, connect!)

  note right of SourceSelected
    Preview line: source port → ray end
    Snap highlight on compatible target port
  end note

旧方式（ドラッグ）→ 新方式（2クリック）に変更済み。ポートの正確な狙いは不要、ノードに当てるだけで最近接ポートを自動選択。
